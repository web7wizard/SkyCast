# 📖 Introduction

Weather applications often focus only on displaying temperature and basic forecasts.

**SKYCAST** reimagines the weather experience by combining:

* Real-time weather monitoring
* Atmospheric analytics
* Air quality tracking
* Interactive weather maps
* Modern glassmorphism UI
* Dynamic weather-based visual effects

The platform provides users with an intuitive dashboard for understanding weather conditions beyond simple forecasts.

```
const API_KEY = "YOUR_API_KEY";
```

# 🚀 Installation Guide

## Prerequisites

* VS Code
* Live Server Extension
* WeatherAPI Key

## Setup

```
git clone https://github.com/yourusername/SKYCAST.git

cd SKYCAST
```

## Configure API Key

Open `script.js` and replace:

```
const API_KEY = "YOUR_API_KEY";
```

with your WeatherAPI key.

## Run

* Open index.html
* Launch with Live Server
